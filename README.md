# Бесконечные Крестики-Нолики (5 в ряд)

## Описание игры

Игра для **двух игроков** на **бесконечном поле**. Побеждает тот, кто первым соберёт **5 одинаковых символов в ряд** (горизонталь, вертикаль, диагональ).

## Архитектура (FSD)

**Особенности:**
- Бесконечная сетка (координаты до `Number.MAX_SAFE_INTEGER`)
- Поочерёдные ходы X → O
- Автоматическое определение победителя
- История всех сыгранных матчей с перемоткой ходов
- UI на TailwindCSS (я плох в дизайне и использовал ai для него)
- Архитектура Feature-Sliced Design (FSD)
- TypeScript + React 19 + Redux Toolkit + Vite
- Unit тесты (70% покрытие)
- Docker контейнеризация

## Быстрый старт

### Локально (npm)
npm install
npm run dev

Открыть: http://localhost:5173

### Docker
docker compose up --build
Открыть: http://localhost:3000

### Тестирование
npm test            


### Production build
npm run build
npm run preview

**Функции:**
-  Список всех сохранённых матчей
-  **Перемотка ходов** (шаг назад/вперёд)
-  Итоговое поле матча
-  Данные: имена, дата, победитель
